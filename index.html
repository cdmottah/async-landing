Acerca del Proyecto

Este proyecto es parte del curso de Asincronismo con JavaScript de Platzi, donde exploramos conceptos como:
• Callbacks
• Promesas
• Async/Await
utilizando la FakeStore API como herramienta de práctica.

Este proyecto demuestra la implementación práctica de conceptos asíncronos en JavaScript, consumiendo datos de una API externa y renderizándolos en una interfaz de usuario moderna. La aplicación utiliza conceptos como fetch API, manejo de errores asíncronos, y manipulación del DOM.
